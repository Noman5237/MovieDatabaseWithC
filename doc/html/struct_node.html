Detailed Description

Node is a core data structure which holds data and pointers to its neighboring nodes.

Conventionally this Node is more commonly known as Doubly since it contains pointer references to both its previous and next neighboring node.

Attention
Do not modify the Node data or pointer references to neighboring nodes unless you know what you doing and are solely responsible for unwanted and weird behaviour of your programs.
