node_t top ( Stack *  stack)

Get the latest available pushed data in the provided stack.

Parameters
stackPointer to the stack instantiated by init()
Exceptions
INVALID_POINTERif stack is either NULL, uninitialized or unaddressable.
INDEX_OUT_OF_BOUNDif stack is empty.
Returns
Latest available pushed data in the provided stack.

Complexity O(1)

Example

#include<DataStruct/stack.h>
int main() {
// Initializing an empty stack
Queue *stack = init();
// Pushing data into the stack
push(stack, 37);
push(stack, 52);
// Getting the top data in the stack
printf("%d\n", top(stack));
// Destroying stack
destroy(stack);
return 0;
}
