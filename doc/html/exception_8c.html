Initiates a pseudo exception handling process with the help of provided trace and handler.

exception_new prints the trace of the caller function and passes the errorCode to the errorCode to handler function.
Depending on the response sent by the handler, it will act accordingly.

Parameters
errorCodeError Code for new exception thrown
fileFile Location of caller function
lineLine Location of caller function
handlerHandler is a callback function for handling this exception and it must return a response
Returns
errorCode if handler responds with anything other than FATAL.

int defaultExceptionHandler ( int  errorCode)

Default exception handler for the library functions.

By default it responds with VERBOSE and FATAL for all known error codes described. It also prints some hints regarding the cause of exception.

To change the default behaviour of the function use setDefaultExceptionHandlerFatality to suit your need and style.
