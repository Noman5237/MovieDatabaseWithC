typedef LinkedList Queue

Queue is nothing special than a one dimensional list which is modified and accessed in unique fashion.

The type Queue is just a syntactic sugar which makes code more meaningful and understandable in contexts of Queue.

Note
As Linked List and Queue shares many functions, so this design pattern was adopted to give users a taste of inheritance like features although it was not the intention at first.
Attention
Do not modify the ll_size of head of the Queue unless you know what you doing and are solely responsible for unwanted and weird behaviour of your programs.
As Queue is a syntactic sugar, so it recommended to use the following functions in order to avoid confusions in contexts of Queue:
ll_init() front() back() ll_forEach() ll_traverse() ll_size() ll_empty() enqueue() dequeue() ll_destroy()
Warning
Any type of modification of Queue without the supplied modifier functions is not intended and the supplied software is not responsible for any fatal or non crashes and unwanted and weird behaviour of the user programs.

node_t front ( Queue *  queue)

Get the earliest available enqueued data of the provided queue.

Parameters
queuePointer to the queue instantiated by ll_init()
Exceptions
INVALID_POINTERif queue is either NULL, uninitialized or unaddressable.
INDEX_OUT_OF_BOUNDif queue is ll_empty.
Returns
Earliest available enqueued data of the provided queue.

Complexity O(1)

Example

#include<DataStruct/queue.h>
int main() {
// Initializing an ll_empty queue
Queue *queue = ll_init();
// Enqueuing data to queue
enqueue(queue, 6);
enqueue(queue, 2);
enqueue(queue, 3);
enqueue(queue, 7);
// Getting the earliest available enqueued data
printf("%d\n", front(queue));
// Jumbling queue
dequeue(queue);
dequeue(queue);
dequeue(queue);
dequeue(queue);
enqueue(queue, 37);
enqueue(queue, 5);
// Getting the earliest available enqueued data
printf("%d\n", front(queue));
// Destroying queue
ll_destroy(queue);
return 0;
}
